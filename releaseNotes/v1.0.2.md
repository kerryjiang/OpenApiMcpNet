# Release Notes

## v1.0.2 (2026-04-13)

### New Features

- **Enum Support**: Added support for enum parameters in OpenAPI specifications, including proper enum value extraction and validation
- **.NET 10 Support**: Added `net10.0` as a target framework alongside `net8.0` and `net9.0`

### Bug Fixes

- Fixed enum value extraction bug in `OpenApiTool` when processing OpenAPI parameters with enum constraints

### Tests

- Added unit and end-to-end tests for enum parameter handling in `OpenApiTool` and `WebApiCaller`

### Documentation

- Updated README with latest features
