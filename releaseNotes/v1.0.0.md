# Release Notes

## v1.0.0 (2026-01-29)

### Initial Release

OpenApiMcpNet is a .NET library that automatically generates Model Context Protocol (MCP) tools from OpenAPI specifications, enabling AI assistants and LLMs to interact with any REST API.

### Features

- **Automatic Tool Generation**: Converts OpenAPI operations into MCP-compatible tools automatically
- **Full OpenAPI Support**: Handles path, query, header, and cookie parameters, as well as request bodies
- **Authentication Support**:
  - OAuth 1.0a (HMAC-SHA1, HMAC-SHA256, PLAINTEXT signature methods)
  - OAuth 2.0 (client credentials grant flow)
  - Extensible authentication via `IAuthenticationHandler` interface
- **Fluent API**: Simple, chainable configuration via `IMcpServerBuilder` extensions
- **Multi-targeting**: Supports .NET 8.0 and .NET 9.0

### API

- `WithToolsFromOpenApi(string openApiSpec, string baseUrl)` - Register tools from OpenAPI JSON/YAML string
- `WithToolsFromOpenApi(Stream openApiSpecStream, string baseUrl)` - Register tools from OpenAPI stream
- `WithToolsFromOpenApi(OpenApiDocument openApiDocument, string baseUrl)` - Register tools from parsed OpenAPI document

### Authentication Handlers

- `OAuth2AuthenticationHandler` - OAuth 2.0 Bearer token authentication
- `OAuth1AuthenticationHandler` - OAuth 1.0a authentication
- `NoOpAuthenticationHandler` - No authentication (for public APIs)

### Dependencies

- ModelContextProtocol 0.5.0-preview.1
- Microsoft.OpenApi.Readers 1.6.28
